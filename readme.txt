Here is a brief description of the Python scripts provided:

Part 1 (Digital Twin): includes the file to filter EMG data in Matlab and filter angles on python. The files can also create .mot files and synchronize IMU and EMG signals.

Part 2 (RNN prediction): includes the files to filter EMG data in Matlab and filter angles on python, the RNN notebooks, the optimization codes (Optuna), codes to synchronize IMU and EMG signals and to visualize them.

The environment used is: GPU_env.yml
